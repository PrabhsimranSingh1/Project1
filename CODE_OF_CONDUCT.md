We, as team members, ensure that we will participate in this project collaboratively and professionally.

We all agree to be respectful towards each other, allow communication, and we all will be open to give and accept feedbacks from each other. The agreed procedures will be followed. 
We will not show any type of disrespectful behaviour towards anyone. No will be harassed, insulted or discriminated.

All members are responsible for following this code of conduct.

If any team member feels awkward or uncomfortable for any reason, they can freely communicate with the scrum master.
This Code of Conduct applies to GitHub (issues, pull requests, comments, commits), Team meetings and any communication related to this project.
